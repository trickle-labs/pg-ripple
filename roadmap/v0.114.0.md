# v0.114.0 — A16 Medium: Module Splits and Architecture Debt

**Status:** ✅ Complete

## Summary

v0.114.0 addresses six medium-severity module-size findings from Assessment 16
(H16-06a, H16-06b, M16-14 through M16-18). Every module exceeding 1,000 LOC is
decomposed into focused sub-modules following the patterns already established in
`src/datalog/` and `src/sparql/`. A CI gate is added to prevent future monolith
growth. An architecture dependency graph is added to `docs/src/architecture.md`.
