# v0.122.0 — A17 God-Module Decomposition & Test Coverage Closure

**Status:** ✅ Released

## Summary

v0.122.0 decomposes all eight remaining god-modules identified in Assessment 17
(H17-02) and closes the test coverage gap for all v0.119.0–v0.120.0 features.

After all splits, zero source files exceed 1,000 LOC. The CI file-size lint gate
is tightened from 1,800 to 1,000 lines. Five new pg_regress test suites are added
covering diagnostic snapshot, read-replica routing, `compat_check()` JSON schema,
tenant quota, and module-split regression testing.
