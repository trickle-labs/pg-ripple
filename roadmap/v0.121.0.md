# v0.121.0 — A17 Security Hardening & Bug Remediation

**Status:** ✅ Released

## Summary

v0.121.0 closes the two High security findings from Assessment 17 and all
medium/low bug and security items from §1–§2 of PLAN_OVERALL_ASSESSMENT_17.md.

The headline fix is replacing the naive string-contains SSRF guard in
`subscribe_rule_library()` with the battle-tested
`resolve_and_check_endpoint()` function (H17-01/SEC-H-01), which performs
proper DNS resolution and validates all resolved IP addresses against the full
SSRF blocklist. The blocklist itself is expanded with CGNAT
(`100.64.0.0/10`), IPv4 multicast (`224.0.0.0/4`), this-network
(`0.0.0.0/8`), and IPv4-mapped IPv6 (`::ffff:0:0/96`) ranges (SEC-M-03).
