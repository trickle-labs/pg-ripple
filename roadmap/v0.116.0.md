# v0.116.0 — A16 Medium: Correctness, Security GUCs and CHANGELOG Hygiene

**Status:** ✅ Released

## Summary

v0.116.0 addresses the remaining medium-severity findings from Assessment 16:
ER monitoring retention (M16-01), rule explanation cache invalidation (M16-05),
cargo-audit review scheduling (M16-06), proof-tree depth/node guards (M16-07),
vocabulary bundle test coverage (M16-08), bidi relay overflow policy (M16-11),
bounded rule explanation LRU (M16-19), Bayesian propagation depth GUC (M16-20),
audit.toml lifecycle policy (M16-21), and CHANGELOG patch entries (M16-23).
