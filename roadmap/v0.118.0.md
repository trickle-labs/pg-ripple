## Summary

v0.118.0 introduces three new features from the Platform Maturity roadmap:
a `pg_ripple.compat_check()` SQL function for belt-and-suspenders version verification
(Feature 3), the seven Allen's interval relations as SPARQL FILTER functions and
Datalog built-ins (Feature 4), and a per-dataset differential privacy budget
registry with automatic daily reset (Feature 2). Also includes AT TIME ZONE support
for temporal queries and an integrated benchmark runner (Feature 1).
